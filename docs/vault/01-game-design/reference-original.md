# 元ゲーム観察メモ — JUNGLE PANG

> 出典: [applion.jp の JUNGLE PANG ページ](https://applion.jp/JUNGLE-PANG/android-com.linefriends.LFJGPAN/)
> 直接プレイしての観察ではなく、ストアページ／レビュー要約からの推定を含む。

## ストアページから読み取れる仕様

- パブリッシャー: LINE Friends
- 主要キャラ: JUNGLE BROWN（LINE Friends 系列の動物キャラ）
- ジャンル: 左右タップアクション、シンプルアーケード
- 評価: ★2.6 / 100,000+ ダウンロード
- 操作: 「縦一列に並んだキャラに対して、一番下のイラストと同じボタンをタップ」
- 難易度: 進行に応じてテンポ加速

## レビューから推測できる UX

- 中毒性の高さは肯定的に評価
- 単純さゆえの飽きと、加速時の追従困難が否定的評価
- フリーズ等の技術的不具合の指摘あり

## 本プロジェクトでの解釈

- メカニクス再現は左右 2 ボタン × 色マッチで MVP として十分
- 「飽き」対策として、後段で色数増加・コンボボーナスを検討（[[difficulty|難易度カーブ]]）
- ビジュアルは独自意匠を必須化（[[../05-assets/asset-list|アセット]]）
