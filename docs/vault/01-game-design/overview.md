# ゲーム概要

## コアフィクション

中央に積み上がる 4 種類のノーツを、対応するキー (S / D / J / K) で達磨落とし式に下から消していく反射アクション。誤ったキーを押すとスコアが減算される。60 秒でどれだけスコアを稼げるか競う。

## ターゲット

- 隙間時間に 1 ラウンド遊べるカジュアル層
- ハイスコアを伸ばしたい反射神経・短期集中型ハイスコア勢

## ワンセンテンス

> 「縦に積み上がる 4 種類のノーツを、対応する S/D/J/K キーで下から達磨落とし式に消し、60 秒で何点取れるか競うキーボードアクション。」
