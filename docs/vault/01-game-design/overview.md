# ゲーム概要

## コアフィクション

中央に積み上がるノーツを、対応するキーで達磨落とし式に下から消していく反射アクション。**2キーモード (D/J) と 4キーモード (S/D/J/K) のいずれかを選択**できる。誤ったキーを押すとスコアが減算される。60 秒でどれだけスコアを稼げるか競う。ランキングはモード別に分離されている。

## ターゲット

- 隙間時間に 1 ラウンド遊べるカジュアル層 (2キー入門可)
- ハイスコアを伸ばしたい反射神経・短期集中型ハイスコア勢 (4キー本気モード)

## ワンセンテンス

> 「縦に積み上がるノーツを、2キー (D/J) または 4キー (S/D/J/K) で下から達磨落とし式に消し、60 秒で何点取れるか競うキーボードアクション。」
