# 2026-05-04 キックオフ

## ゴール

LINE Friends の Android ゲーム「JUNGLE PANG」を参考にした Web ゲーム（コードネーム **JUNGLE TAP**）の開発を開始する。

## 決定事項

| 項目 | 決定 | 代替案・備考 |
|------|------|--------------|
| 技術スタック | Web (HTML5 + Phaser 3 + TypeScript + Vite) | Unity / Godot / Flutter+Flame |
| プラットフォーム | Web（PC / モバイルブラウザ） | Android / iOS は将来検討 |
| オリジナル要素 | ほぼ忠実なクローン（メカニクス再現、ビジュアルは独自意匠） | 大幅アレンジ |
| ノート管理 | プロジェクト内に Obsidian vault (`docs/vault/`) | 別リポジトリ・別ディレクトリ |
